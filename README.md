# Function-Plotter






<h2> Dependencies </h2>

<p> pip install PySide2 </p>
<p> pip install numpy </p>
<p> pip install matplotlib </p>

<p> the function is of one variable x </br>

<h2> supported functions: </h2>

<h3> x^n </h3>
<h3> e^(x) </h3>
<h3> ln(x)</h3>
<h3> sin(x)</h3>
<h3> cos(x)</h3>
<h3> tan(x) </h3>
