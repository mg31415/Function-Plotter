# Function-Plotter






Dependencies

pip install PySide2
pip install numpy
pip install matplotlib

the function is of one variable x

supported functions:

<h1> x^n </h1>
<h1> e^(x) </h1>
<h1> ln(x)</h1>
<h1> sin(x)</h1>
<h1> cos(x)</h1>
<h1> tan(x) </h1>
